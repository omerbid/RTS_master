
# RTS Project Overview

This project is a Hero‑centric RTS built in Unreal Engine 5.

Core idea:
A living world where regions simulate population, wildlife, economy,
and conflict continuously on the same map used for tactical battles.

Factions:
- Humans
- Vampires
- Werewolves

Key pillars:
1. Living World Simulation
2. Region Control instead of base building
3. Hero‑centric command structure
4. Ecological economy (population + wildlife)

Cursor rule:
Never rewrite working gameplay systems unless explicitly instructed.
Extend systems instead of replacing them.
